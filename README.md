# FluidCursor
FluidCursor, is a small **cursor-effect app** for macos with little resource usages!  And with a usefull "cmd+tab" function to reopen minimized window. 


| effects | English Translation | marks / 常见场景 |
| --- | --- | --- |
| **流体** | Fluid | 液体、烟雾等流体动力学效果 |
| **普通线条** | Basic Lines / Normal Lines | 基础线段或轨迹线 |
| **水波（扩散）** | Water Ripple (Diffusion) / Ripple Expansion | 从中心向外扩散的水波 |
| **水波折射（需屏幕录制）** | Water Refraction (Screen Recording Required) | 带有画面扭曲、折射效果的动态水波 |
| **水波纹（圆环）** | Circular Ripple / Water Ring | 呈现完美圆环状的水面波纹 |
| **呼吸光晕** | Breathing Glow / Pulsing Aura | 像呼吸一样忽明忽暗、渐大渐小的光圈 |
| **蓄力（闭合环）** | Charging (Closed Ring) / Closed Loop Charge | 能量向内收缩、闭合的环状蓄力特效 |
| **蓄力（能量环）** | Charging (Energy Ring) / Energy Loop | 带有粒子或光效的能量聚集环 |
